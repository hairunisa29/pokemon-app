# Pokemon App

This is a simple pokemon app for catching pokemon and save it to your own collection.

### Built With

* React

## Getting Started

### Installation
1. Clone the repo
   ```sh
   git clone https://github.com/hairunisa29/pokemon-app.git
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
4. Run the app
   ```sh
   npm run dev
   ```
